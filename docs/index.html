Features

Everything You Need

Simple yet powerful features to manage your game saves

Automatic Snapshots

Create timestamped backups of your save files with a single click. Keep multiple versions organized.

Instant Restore

Switch between saves instantly. Perfect for trying different game paths or recovering from mistakes.

Smart Protection

Automatically backs up current save before restoring. Never lose progress accidentally.

Game Covers

Upload custom cover images for your games. Beautiful visual organization.

Cross-Platform

Available for Windows, Linux, and macOS. Your saves go where you go.

Process Detection

Prevents restore while game is running. Smart safety checks protect your data.

Simple

How It Works

Get started in three easy steps

1

Add Your Game

Tell Checkpoint where your save files are located. Add an optional cover image for easy identification.

2

Create Snapshots

Click "Create Snapshot" anytime to save your current progress. Name it whatever you want.

3

Restore Anytime

Select any snapshot and click restore. Your save is instantly back to that point.

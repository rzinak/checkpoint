Features

Everything You Need

Powerful features designed to protect your game saves with complete privacy

One-Click Snapshots

Create timestamped backups instantly. Name them "Before Boss Fight" or "Good Ending" to remember exactly what they contain.

Works 100% Offline

No internet required. All features work locally on your computer. Cloud backup is completely optional.

Google Drive Backup

Optionally back up saves to your personal Google Drive. Access from any device. Cloud is entirely optional.

Instant Restore

Switch between saves in seconds. Perfect for trying different game paths or recovering from mistakes.

Smart Protection

Automatically backs up current save before restoring. Never lose progress accidentally.

Cross-Platform & Open Source

Use on Windows and Linux. Full transparency with MIT license - check the code, contribute, or fork it.

Privacy First

We Can't See Your Saves.
Even If We Wanted To.

Checkpoint is built with privacy as the foundation. Your game saves are yours alone.

No Account Needed

Use Checkpoint locally without creating any account. No email, no password, no tracking.

Zero Server Access

We don't have servers. Your saves never pass through our infrastructure. Cloud backups go directly to Google.
